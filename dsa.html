Singly Linked List Insertion

A singly linked list is a data structure where each node contains data and a pointer to the next node. Here's an implementation of inserting a node at the front of a singly linked list.

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include <iostream>
using namespace std;

class Node {
public:
    int data;
    Node* next;

    Node(int value) {
        data = value;
        next = nullptr;
    }
};

class SinglyLinkedList {
private:
    Node* head;

public:
    SinglyLinkedList() { head = nullptr; }

    void insertAtFront(int value) {
        Node* newNode = new Node(value);
        newNode->next = head;
        head = newNode;
    }

    void display() {
        Node* temp = head;
        while (temp != nullptr) {
            cout << temp->data << " ";
            temp = temp->next;
        }
        cout << endl;
    }
};

int main() {
    SinglyLinkedList list;

    list.insertAtFront(4);
    list.insertAtFront(3);
    list.insertAtFront(2);

    cout << "Original linked list: ";
    list.display();

    list.insertAtFront(1);

    cout << "After inserting node at the front: ";
    list.display();

    return 0;
}

Output:

Original linked list: 2 3 4
After inserting node at the front: 1 2 3 4

Stack Operations

A stack is a linear data structure that follows the Last In First Out (LIFO) principle. Here's an implementation of basic stack operations including push, pop, and isEmpty.

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include <iostream>
using namespace std;

class Stack {
private:
    int top;
    int arr[100];

public:
    Stack() {
        top = -1;
    }

    void push(int x) {
        if (top >= 99) {
            cout << "Stack overflow" << endl;
            return;
        }
        arr[++top] = x;
        cout << "Pushed " << x << " to stack\n";
    }

    int pop() {
        if (top < 0) {
            cout << "Stack underflow" << endl;
            return 0;
        }
        return arr[top--];
    }

    bool isEmpty() {
        return (top < 0);
    }
};

int main() {
    Stack s;
    s.push(10);
    s.push(20);
    s.push(30);

    cout << "Elements present in stack: ";

    while (!s.isEmpty()) {
        cout << s.pop() << " ";
    }
    
    return 0;
}

Output:

Pushed 10 to stack
Pushed 20 to stack
Pushed 30 to stack
Elements present in stack: 30 20 10

Queue Operations

A queue is a linear data structure that follows the First In First Out (FIFO) principle. Here's an implementation of basic queue operations including enqueue, dequeue, and display.

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include <iostream>
using namespace std;

class Queue {
private:
    int* arr;
    int front, rear, size, capacity;

public:
    Queue(int size) {
        capacity = size;
        arr = new int[capacity];
        front = rear = -1;
    }

    bool isEmpty() {
        return rear == -1;  
    }

    bool isFull() {
        return rear == capacity - 1;
    }

    void enqueue(int item) {
        if (isFull()) { 
            cout << "Queue Overflow! cannot enqueue\n"; 
            return;
        }
        if (isEmpty()) {
            front = 0;
        }
        rear++;
        arr[rear] = item;
        cout << "Enqueued: " << item << endl;
    }

    void dequeue() {
        if (isEmpty()) {
            cout << "Queue Underflow! cannot dequeue\n"; 
            return;
        }
        cout << "Dequeued: " << arr[front] << endl;
        if (front == rear) { 
            front = rear = -1;
        } else {
            front++;
        }
    }

    void display() {
        if (isEmpty()) {
            cout << "Queue is Empty\n";
            return;
        }
        cout << "Queue elements: ";
        for (int i = front; i <= rear; i++) { 
            cout << arr[i] << " ";
        }
        cout << endl;
    }
};

int main() {
    Queue q(5);
    q.enqueue(10);
    q.enqueue(20);
    q.enqueue(30);
    q.display();
    q.dequeue();
    q.display();
    q.enqueue(40);
    q.enqueue(50);
    q.enqueue(60);
    q.display();

    return 0;
}

Binary Search

Binary search is an efficient algorithm for searching a sorted array by repeatedly dividing the search interval in half. It has a time complexity of O(log n).

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include<iostream>
using namespace std;

int main() {
    int i, arr[10], num, first, last, middle;
    cout << "Enter 10 elements (in ascending order): ";
    for(i = 0; i < 10; i++)
        cin >> arr[i];
    cout << "\n Enter element to be search: ";
    cin >> num;
    first = 0;
    last = 9;
    middle = (first + last)/2;
    while(first <= last)
    {
        if(arr[middle] < num)
            first = middle + 1;
        else if(arr[middle] == num)
        {
            cout << "\n Element found at position " << middle + 1;
            break;
        }
        else
            last = middle - 1;
        middle = (first + last)/2;
    }
    if(first > last)
        cout << "\n Element not found!";
    return 0;
}

Heap Sort

Heap sort is a comparison-based sorting algorithm that uses a binary heap data structure. It has a time complexity of O(n log n) for all cases.

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include <iostream>
using namespace std;

void heapify(int arr[], int n, int i) {
    int largest = i; 
    int left = 2 * i + 1; 
    int right = 2 * i + 2; 

    // If left child is larger than root
    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }

    // If right child is larger than largest so far
    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }

    // If largest is not root
    if (largest != i) {
        swap(arr[i], arr[largest]);

        // Recursively heapify the affected sub-tree
        heapify(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    // Build heap (rearrange array)
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }

    // One by one extract an element from heap
    for (int i = n - 1; i > 0; i--) {
        // Move current root to end
        swap(arr[0], arr[i]);

        // Call max heapify on the reduced heap
        heapify(arr, i, 0);
    }
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr[] = {12, 11, 13, 5, 6, 7};
    int n = sizeof(arr) / sizeof(arr[0]);

    cout << "Unsorted array: ";
    printArray(arr, n);

    heapSort(arr, n);

    cout << "Sorted array: ";
    printArray(arr, n);

    return 0;
}

Output:

Unsorted array: 12 11 13 5 6 7
Sorted array: 5 6 7 11 12 13

Quick Sort

Quick sort is a divide-and-conquer algorithm that works by selecting a 'pivot' element and partitioning the array around the pivot. It has an average time complexity of O(n log n).

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include <iostream>
using namespace std;

void swap(int *a, int *b) {
    int t = *a;
    *a = *b;
    *b = t;
}

void printArray(int array[], int size) {
    int i;
    for (i = 0; i < size; i++)
        cout << array[i] << " ";
    cout << endl;
}

int partition(int array[], int low, int high) {
    int pivot = array[high];
    int i = (low - 1);

    for (int j = low; j < high; j++) {
        if (array[j] <= pivot) {
            i++;
            swap(&array[i], &array[j]);
        }
    }
    
    swap(&array[i + 1], &array[high]);
    return (i + 1);
}

void quickSort(int array[], int low, int high) {
    if (low < high) {
        int pi = partition(array, low, high);
        quickSort(array, low, pi - 1);
        quickSort(array, pi + 1, high);
    }
}

int main() {
    int data[] = {8, 7, 6, 1, 0, 9, 2};
    int n = sizeof(data) / sizeof(data[0]);
    
    cout << "Unsorted Array: \n";
    printArray(data, n);
    
    quickSort(data, 0, n - 1);
    cout << "Sorted array in ascending order: \n";
    printArray(data, n);

    return 0;
}

Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. It has a time complexity of O(nÂ²).

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include <iostream>
using namespace std;

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);

    cout << "Unsorted array: ";
    printArray(arr, n);

    bubbleSort(arr, n);

    cout << "Sorted array: ";
    printArray(arr, n);

    return 0;
}

Inorder Traversal

Inorder traversal is a way of traversing a binary tree where we visit the left subtree, then the root, and finally the right subtree (Left â†’ Root â†’ Right). This traversal gives nodes in non-decreasing order for a Binary Search Tree.

Important Note: This implementation is directly provided by the faculty as part of the official course material. These codes are authentic course content and are not generated using ChatGPT or any other AI tools. All code examples and implementations are verified and approved by the course instructors.

C++ Implementation 
#include<iostream>
using namespace std;

class Node {
    public:
    int data;
    Node* left;
    Node* right;

    Node(int data) {
        this->data = data;
        this->left = NULL;  
        this->right = NULL;
    }
};

class BinaryTree {
    public:
    Node* root;

    BinaryTree() { root = NULL; }

    void inorderTraversal(Node* node) {
        if (node != nullptr) {
            inorderTraversal(node->left);
            cout << node->data << " ";

Frequently Asked Questions

What are the most important data structures to learn?

The most essential data structures include arrays, linked lists, stacks, queues, trees (especially binary trees), graphs, and hash tables. These form the foundation of most programming solutions and algorithms.

Which sorting algorithm is the most efficient?

The efficiency of sorting algorithms depends on the specific use case. Quick Sort and Merge Sort are generally the most efficient with average time complexity of O(n log n). However, for small datasets, simpler algorithms like Insertion Sort might perform better due to lower overhead.

How do I prepare for DSA interviews?

Focus on understanding the fundamentals of each data structure and algorithm rather than memorizing implementations. Practice solving problems on platforms like LeetCode, HackerRank, and CodeForces. Explain your thought process while solving problems, as interviewers value your approach as much as the final solution.

What's the difference between inorder, preorder, and postorder traversal?

These are different ways to traverse a binary tree. Inorder traversal visits the left subtree, then the root, then the right subtree (Left-Root-Right). Preorder traversal visits the root first, then the left and right subtrees (Root-Left-Right). Postorder traversal visits the left and right subtrees before the root (Left-Right-Root).

When should I use a linked list instead of an array?

Use linked lists when you need frequent insertions and deletions, especially at the beginning or middle of the list. Linked lists provide O(1) insertion and deletion at known positions. Arrays are better when you need random access to elements or when memory efficiency is crucial.
